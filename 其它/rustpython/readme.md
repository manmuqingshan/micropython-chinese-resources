# RustPython

使用 Rust 编写的极简安全的 Python 解释器 monty，供 AI 使用。

monty 避免了使用基于完整容器的沙盒来运行生成的LLM代码所带来的成本、延迟、复杂性和普遍负担。

相反，它允许你安全地运行由嵌入在代理中的LLM编写的Python代码，启动时间以个位数微秒为单位，而不是数百毫秒。

**monty 能做什么**:
- 运行一个合理的Python代码子集 —— 足以让你的代理表达其想要做什么
- 完全阻止对主机环境的访问:文件系统、env 变量和网络访问均通过开发者可以控制的外部函数调用实现
- 主机上的调用函数——仅允许其访问的函数
- 运行类型检查 - monty 支持完整的现代 Python 类型提示，并附带 ty 包含在单个二进制文件中以运行类型查询
- 在外部函数调用时将解释器状态按字节进行快照，然后稍后恢复
- 启动速度极快(从代码到执行结果),运行时性能与CPython相似(通常速度在5倍快到5倍慢)
- 从 Rust、Python 或 JavaScript 调用 —— 由于 Monty 对 cpython 没有依赖，因此你可以在任何能运行 Rust 的地方使用它
- 控制资源使用情况 —— monty 能够跟踪内存使用情况、分配、堆叠深度和执行时间，并在超过预设限制的情况下取消执行
- 收集 stdout 和 stderr，然后返回给调用者
- 通过主机上的 async 或 sync 在主机上运行代码
