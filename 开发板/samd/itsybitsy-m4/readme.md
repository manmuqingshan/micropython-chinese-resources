# Adafruit ItsyBitsy M4 Express

Adafruit ItsyBitsy M4 Express 体积小，功能强大，使用运行频率为 120 MHz 的 ATSAMD51 Cortex M4 处理器，支持浮点，具有 512KB 闪存和 192KB RAM。
