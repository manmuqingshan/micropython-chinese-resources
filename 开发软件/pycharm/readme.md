# PyCharm

PyCharm 是一个适用于数据科学和 Web 开发的 Python IDE，提供免费的 Community 版本和付费的 Professional 版本。通过 MicroPython 插件，可以在 PyCharm 中轻松使用 MicroPython。

安装插件后，可以进入 `Settings -> Languages & Frameworks -> MicroPython Tools` 启用 micropython 支持。
