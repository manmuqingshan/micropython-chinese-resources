# 打造一个农历新年盒子

这个木盒正面有五颗纽扣，会用普通话、粤语、越南语、韩语或带有美式口音的英语表达农历新年问候。

![](lunar-box.webp)

使用了 Adafruit Feather RP2040 开发板，通过 circuitpython 编程。
